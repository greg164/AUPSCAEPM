Objet n°7 — Memory Jar

TYPE : Artéfact anomal RARETÉ : Très rare DANGER : Psychologique — utilisation à risque
MEMORY JAR — BOCAL STANDARD

Description

Le Memory Jar est un bocal en verre ordinaire — type bocal de confiture — qui se distingue par son contenu : une substance lumineuse légèrement colorée, comme de l'air condensé. Quand on ouvre le bocal et qu'on l'approche du visage, on revit pendant quelques secondes un souvenir du monde réel.

Ces souvenirs sont intenses, sensoriels et précis — une odeur familière, une voix aimée, une sensation tactile. Beaucoup d'explorateurs les considèrent comme les objets les plus précieux des Backrooms, non par leur utilité pratique, mais par leur valeur psychologique dans des moments de désespoir absolu.

Effets secondaires

L'usage répété du même Memory Jar efface progressivement le souvenir qu'il contient — à chaque ouverture, le souvenir est légèrement moins précis. Après 8 à 12 utilisations, le bocal est vide et le souvenir réel de l'explorateur lui-même devient flou. Des explorateurs ont développé des dépendances sévères aux Memory Jars, les ouvrant jusqu'à ne plus se souvenir de rien.

⚠ Le M.E.G. recommande de limiter l'usage à 2 ouvertures maximum par Memory Jar. Ne jamais utiliser un bocal appartenant à quelqu'un d'autre — les conséquences psychologiques sont documentées comme extrêmement perturbantes.
